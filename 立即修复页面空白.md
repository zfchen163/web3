# 🚀 立即修复页面空白问题

---

## ⚡ 快速修复步骤

### 步骤 1：打开浏览器开发者工具

1. 按 `F12` 或 `Cmd+Option+I` (Mac)
2. 点击 **Console** 标签页
3. **复制所有错误信息给我**

---

### 步骤 2：检查常见问题

#### 问题 A：模块未找到

如果看到：
```
Cannot find module 'antd'
Cannot find module 'element-china-area-data'
```

**解决方案：**
```bash
cd /Users/h/practice/chain-vault/frontend
npm install
```

---

#### 问题 B：React 未加载

如果看到：
```
React is not defined
```

**解决方案：**
```bash
cd /Users/h/practice/chain-vault/frontend
npm install react react-dom
```

---

#### 问题 C：端口被占用

如果看到：
```
Port 3000 is already in use
```

**解决方案：**
```bash
# 查找占用端口的进程
lsof -i :3000

# 停止进程（替换 PID 为实际进程号）
kill -9 <PID>

# 或修改端口
# 编辑 vite.config.ts，将 port: 3000 改为 port: 3001
```

---

### 步骤 3：强制刷新浏览器

1. 按 `Cmd+Shift+R` (Mac) 或 `Ctrl+Shift+R` (Windows)
2. 或右键刷新按钮 → "清空缓存并硬性重新加载"

---

### 步骤 4：重启服务器

```bash
# 在运行 vite 的终端按 Ctrl+C 停止
# 然后重新启动
cd /Users/h/practice/chain-vault/frontend
npm run dev
```
