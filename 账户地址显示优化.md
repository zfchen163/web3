### JavaScript 逻辑优化

**修改前**：
```javascript
// 使用 formatAddress 截断地址
document.getElementById('accountAddress').textContent = formatAddress(currentAccount);

// formatAddress 函数
function formatAddress(address, full = false) {
    if (!address) return '';
    if (full) return address;
    return `${address.slice(0, 6)}...${address.slice(-4)}`; // 截断为 0xf39F...2266
}
```

**修改后**：
```javascript
// 直接显示完整地址
document.getElementById('accountAddress').textContent = currentAccount;
```

**改进说明**：
- ✅ 移除地址截断逻辑
- ✅ 直接显示完整的 42 字符地址
- ✅ 保留 `formatAddress` 函数供其他地方使用
