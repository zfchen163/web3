## 🎨 新增功能

### 1. 一键填写按钮 ⭐
```
┌────────────────────────────────────┐
│ 🎲 快速测试                        │
│ 点击按钮自动填充所有表单字段        │
│                                    │
│        [✨ 一键填写所有字段]        │
└────────────────────────────────────┘
```

**功能：**
- 自动生成所有字段的随机测试数据
- 包括品牌、型号、颜色、尺码等
- 自动生成序列号
- 随机决定是否上架和价格

### 2. 自动生成序列号
```
序列号: [NK-SHOES-20240107-1234] [⚡ 自动生成]
```

**功能：**
- 根据品牌和分类自动生成
- 格式：品牌-分类-日期-随机数
- 保证唯一性

### 3. 表单验证
- 实时验证必填字段
- 显示错误提示
- 防止提交不完整的表单

### 4. 图片预览
- 上传后立即预览
- 支持多张图片
- 可以删除已上传的图片

## 🎯 使用方法

### 方式 1：一键填写（推荐用于测试）

1. 访问注册页面
2. 点击"✨ 一键填写所有字段"按钮
3. 所有字段自动填充随机数据
4. 上传图片（这个需要手动）
5. 点击"✅ 注册资产"

### 方式 2：手动填写

1. 逐个填写表单字段
2. 必填字段标有 * 号
3. 使用"⚡ 自动生成"按钮生成序列号
4. 上传商品照片
5. 选择是否立即上架
6. 提交表单
