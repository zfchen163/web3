## 🚀 快速开始

### 第一次使用

1. **访问主页**
   ```
   http://localhost:3000/
   ```

2. **自动跳转到版本选择页面**

3. **选择版本**
   - 推荐：点击"原生 HTML 版本"
   - 或：点击"React 版本"

4. **连接钱包**
   - 点击右上角"连接钱包"
   - MetaMask 确认连接

5. **开始使用**
   - 卖家：注册和上架资产
   - 买家：浏览和购买资产

## 🔧 技术说明

### 实现方式

1. **主入口重定向**
   - `index.html` 自动跳转到 `welcome.html`
   - 使用 JavaScript 重定向

2. **版本选择页面**
   - `welcome.html` 提供两个版本的链接
   - 推荐原生 HTML 版本（高亮显示）

3. **独立版本入口**
   - 原生 HTML：`index-native.html`
   - React 版本：`react.html`
