# ChainVault - 区块链资产交易平台

一个完整的区块链资产注册、验证和交易平台，支持品牌授权、资产上链、二手交易等功能。

## 🎨 全新 UI 版本

我们推出了全新的**原生 HTML + Tailwind CSS + Swiper** 版本！

### ✨ 新版本特色

- 🎨 **现代化渐变设计** - 紫蓝渐变主题
- 💎 **玻璃态效果** - 半透明模糊背景
- 🎬 **流畅动画** - 浮动、旋转、滑入效果
- 🎡 **Swiper 轮播** - 特性展示和图片轮播
- 📱 **响应式布局** - 完美适配所有设备
- ⚡ **快速加载** - 无需构建，直接运行

## 🎯 使用流程

### 1. 连接钱包
```
点击"连接钱包" → MetaMask 授权 → 连接成功
```

### 2. 注册资产
```
填写资产信息 → 上传图片（可选） → 提交到区块链 → 等待确认
```

### 3. 上架资产
```
进入"我的资产" → 选择资产 → 点击"上架" → 设置价格 → 确认
```

### 4. 购买资产
```
浏览市场 → 选择资产 → 查看详情 → 点击"购买" → 确认支付
```

### 5. 订单管理
```
进入"我的订单" → 查看状态 → 执行操作（发货/收货/退款）
```

## 🐛 问题排查

### 页面无法加载
1. 检查服务器是否启动
2. 确认端口号正确（3000 或 3002）
3. 清除浏览器缓存

### 无法连接钱包
1. 确保 MetaMask 已安装
2. 检查 MetaMask 是否解锁
3. 确认连接到正确的网络

### 交易失败
1. 检查账户余额
2. 确认 Gas 费用充足
3. 验证合约地址正确
