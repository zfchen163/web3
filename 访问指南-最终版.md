## 🎯 完整使用流程

### 第一次访问

1. **打开浏览器**
   ```
   http://localhost:3000/
   ```

2. **自动跳转到版本选择页**
   - 看到两个版本选项
   - 原生 HTML 版本（推荐）⭐
   - React 版本

3. **点击"原生 HTML 版本"**
   - 进入主界面
   - 看到精美的渐变背景

4. **连接钱包**
   - 点击"连接钱包"按钮
   - MetaMask 弹出连接请求
   - 确认连接

5. **开始使用**
   - 浏览市场
   - 注册资产
   - 管理订单

---

### 买家购买流程

1. **访问市场**
   ```
   http://localhost:3000/index-native.html
   ```

2. **连接钱包**

3. **浏览商品**
   - 点击"🛍️ 市场"标签
   - 查看在售资产
   - 悬停卡片查看动画效果

4. **查看详情**
   - 点击"查看详情"按钮
   - 800px 宽屏显示
   - 查看完整信息

5. **购买资产**
   - 点击"购买"按钮
   - 确认购买对话框
   - 等待交易确认
   - ✅ 购买成功！

---

### 卖家注册流程

1. **访问注册页**
   ```
   http://localhost:3000/index-native.html
   ```

2. **连接钱包**

3. **注册资产**
   - 点击"➕ 注册资产"标签
   - 看到完整的注册表单

4. **填写信息**
   - **方式 1**：点击"✨ 一键填写所有字段"
   - **方式 2**：手动填写每个字段

5. **上传图片**
   - 选择商品照片
   - 查看预览

6. **提交注册**
   - 点击"✅ 注册资产"
   - 等待交易确认
   - ✅ 注册成功！

7. **管理资产**
   - 点击"📦 我的资产"
   - 查看已注册的资产
   - 上架/下架/转移
