## 🗺️ 2. 省市区选择器

### 功能说明
提供简单的二级联动（省份 → 城市）

### 实现方式
```tsx
// 内置常用省市数据
const regionData: Record<string, string[]> = {
  '北京市': ['东城区', '西城区', '朝阳区', ...],
  '上海市': ['黄浦区', '徐汇区', '长宁区', ...],
  '广东省': ['广州市', '深圳市', '珠海市', ...],
  '浙江省': ['杭州市', '宁波市', '温州市', ...],
  '江苏省': ['南京市', '苏州市', '无锡市', ...],
  // ... 更多省份
}
```

### 界面展示
```
生产日期: [___________]    |    生产地: [省份▼] [城市▼]
```

### 交互流程
1. 用户选择省份 → 自动加载该省的城市列表
2. 用户选择城市 → 自动组合为 "省份 城市"
3. 结果自动填入 `productionLocation` 字段

### 特点
- ✅ **二级联动**：选择省份后自动更新城市列表
- ✅ **自动组合**：系统自动组合完整地址
- ✅ **禁用状态**：未选省份时城市选择器禁用
- ✅ **内置数据**：包含12个主要省份及其城市

## ⚡ 4. 序列号自动生成

### 功能说明
智能生成唯一序列号，也支持手动输入

### 界面展示
```
序列号: [NK-SHOES-20240107-8234] [⚡ 自动生成]
        ↑ 输入框                    ↑ 生成按钮
```

### 生成规则
```
格式：品牌缩写-分类-日期-随机数

示例：
- Nike 鞋类 → NK-SHOES-20240107-8234
- Adidas 服装 → AD-CLOTH-20240107-5621
- Supreme 配饰 → SUP-ACCES-20240107-3489
```

### 生成逻辑
```typescript
function generateSerialNumber() {
  // 1. 品牌缩写（前2-3个字母）
  const brandPrefix = "NK";  // Nike → NK
  
  // 2. 分类名称
  const category = "SHOES";   // 鞋类 → SHOES
  
  // 3. 日期（年月日）
  const date = "20240107";    // 2024年1月7日
  
  // 4. 随机4位数
  const random = "8234";      // 1000-9999
  
  // 组合
  return "NK-SHOES-20240107-8234";
}
```

### 特点
- ✅ **智能生成**：基于品牌和分类自动创建
- ✅ **唯一性**：包含日期和随机数，确保唯一
- ✅ **可手动**：也可以手动输入自定义序列号
- ✅ **大写转换**：自动转换为大写字母
- ✅ **格式验证**：只允许字母、数字和连字符

### 按钮样式
- **渐变背景**：绿色渐变（#11998e → #38ef7d）
- **悬浮效果**：鼠标悬停时上浮和阴影增强
- **明显标识**：⚡ 图标 + "自动生成" 文字

### 4. 序列号生成

```tsx
// 生成按钮
<button onClick={generateSerialNumber}>
  ⚡ 自动生成
</button>

// 生成逻辑
function generateSerialNumber() {
  const brand = formData.brand.substring(0, 3).toUpperCase();
  const category = categoryMap[formData.category];
  const date = formatDate(new Date());
  const random = Math.floor(1000 + Math.random() * 9000);
  
  return `${brand}-${category}-${date}-${random}`;
}
```
