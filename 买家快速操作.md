# 🛒 买家快速操作指南

## 🚀 3 步完成购买

### 第 1 步：连接钱包
```
1. 访问：http://localhost:3000/index-native.html
2. 点击右上角"连接钱包"
3. MetaMask 确认连接
```

### 第 2 步：浏览市场
```
1. 默认显示"市场"页面
2. 查看所有在售资产
3. 点击"查看详情"查看完整信息
```

### 第 3 步：购买资产
```
1. 点击资产卡片的"购买"按钮
2. MetaMask 确认交易
3. 等待区块链确认
4. 购买成功！
```

## 📋 订单管理

### 订单状态流程
```
已支付 → 已发货 → 已送达 → 已完成
```

### 买家操作

| 订单状态 | 买家操作 | 说明 |
|---------|---------|------|
| 已支付 | 等待发货 | 卖家需要点击"发货" |
| 已发货 | 确认收货 | 收到商品后点击 |
| 已送达 | 完成交易 | 确认无误后点击 |
| 任何状态 | 申请退款 | 符合条件时可申请 |

## ❓ 常见问题

### Q: 如何看到卖家上架的资产？
**A:** 连接钱包后，点击"市场"标签，所有在售资产会自动显示。

### Q: 地址显示不完整怎么办？
**A:** 点击"查看详情"按钮，在详情页面可以看到完整地址，并可以复制。

### Q: 购买需要多少 ETH？
**A:** 资产价格 + Gas 费（约 0.001-0.01 ETH）

### Q: 购买后多久能收到？
**A:** 
- 链上交易：立即完成（几秒）
- 实物交付：等待卖家发货

### Q: 如何申请退款？
**A:** 在"我的订单"页面，点击"申请退款"按钮。
