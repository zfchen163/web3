## 🎯 使用流程

### 1. 连接钱包

1. 打开前端：http://localhost:5173
2. 点击"连接钱包"
3. MetaMask 弹出，点击"连接"
4. 连接成功，显示账户地址

### 2. 注册资产

1. 点击"➕ 注册资产"标签
2. 填写资产信息：
   - 资产名称：耐克 Air Jordan 1 红黑配色
   - 序列号：NK-AJ1-2024-001
   - 元数据 URI：（可选）
3. 点击"注册资产"
4. MetaMask 弹出，确认交易
5. 等待确认，注册成功！

### 3. 上架销售

1. 点击"📦 我的资产"标签
2. 找到刚注册的资产
3. 点击"上架"按钮
4. 输入价格：0.5（ETH）
5. 确认交易
6. 上架成功！

### 4. 购买资产

1. 切换到另一个账户（MetaMask）
2. 点击"🛒 市场"标签
3. 找到想要的商品
4. 点击"购买"按钮
5. 确认支付
6. 购买成功！

### 5. 完成交易

1. 卖家点击"发货"
2. 买家点击"确认收货"
3. 等待3天退货期
4. 点击"完成交易"
5. 资金支付给卖家
6. 所有权转移给买家

## 🆘 常见问题

### Q1: 连接钱包失败？

**A**: 检查 MetaMask 是否安装，并切换到 Hardhat Local 网络

```
网络名称：Hardhat Local
RPC URL：http://127.0.0.1:8545
Chain ID：31337
货币符号：ETH
```

### Q2: 交易失败？

**A**: 检查：
1. 账户是否有足够的 ETH
2. 是否使用了正确的账户（品牌方/所有者）
3. 资产/订单状态是否正确

### Q3: 后端连接失败？

**A**: 检查：
1. MySQL 是否运行
2. 数据库是否已创建
3. .env 配置是否正确

### Q4: 前端显示空白？

**A**: 检查：
1. 是否运行了 `npm run dev`
2. 浏览器控制台是否有错误
3. 合约地址是否正确
